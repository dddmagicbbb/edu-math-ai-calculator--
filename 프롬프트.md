# Role
너는 교육 공학(EdTech) 분야의 숙련된 시니어 소프트웨어 엔지니어이자 기술 작가(Technical Writer)야. 내가 제공하는 프로젝트 아이디어를 바탕으로 전문적인 '프로그램 개요서'를 Markdown(.md) 형식으로 작성해줘.

# Project Overview
- 프로젝트 명: 학습 밀착형 AI 공학계산기 (Edu-Math AI Calculator)
- 핵심 가치: 단순 계산 결과 도출을 넘어, 학습자의 수준에 맞는 수학적 원리와 유사 문제를 제공하여 완전 학습을 지원함.
- 주요 기능:
  1. 수식 계산 및 결과 출력
  2. 단계별 풀이 과정 및 핵심 수학 개념 설명
  3. 학습자 수준별 맞춤 설명 (초등/중등/고등/대학생 난이도 구분)
  4. 학습 강화용 유사 문제 생성

# Tech Stack & Workflow
- Frontend: HTML5, CSS3, JavaScript (Vanilla JS)
- Backend: Python (Flask)
- AI Engine: Groq AI API (수식 해석 및 설명 생성용)
- Infrastructure: Git (Version Control), Vercel (Deployment)
- 데이터 흐름: 사용자가 웹(HTML)에 수식 및 학습 수준을 입력하면, Flask 서버가 Groq API를 호출하여 수준별 맞춤 답변을 생성하고, 이를 다시 화면에 렌더링함.

# Requirements for the Markdown Document
개요서에는 다음 항목들이 반드시 포함되어야 해:
1. 프로젝트 개요 (Introduction): 개발 배경 및 타겟 사용자
2. 핵심 기능 상세 (Key Features): 수준별 설명 로직 및 유사 문제 제공 방식 설명
3. 기술 스택 (Tech Stack): 각 기술을 선택한 이유 요약
4. 시스템 아키텍처 (Architecture): 프론트엔드 - 백엔드 - Groq API 간의 시퀀스 설명
5. 상세 파일 구조 (Project Structure): Vercel 배포를 고려한 Flask 프로젝트 구조
6. UI/UX 가이드라인: 수식 가독성을 위한 LaTeX 적용 및 사용자 경험 설계
7. 향후 확장성: 그래프 시각화 및 학습 이력 저장 기능 등

# Tone & Style
- 전문적이고 체계적인 비즈니스 문체 사용
- 수식은 반드시 LaTeX 문법($...$ 또는 $$...$$)을 사용하여 표기하도록 명시
- Markdown의 표(Table), 리스트, 코드 블록을 활용하여 가시성 극대화
