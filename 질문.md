# Edu-Math AI 프로젝트 질의 내역 (Conversation History)

본 문서는 프로젝트 시작부터 완성까지 사용자가 직접 질의한 내용을 원문 그대로 기록한 문서입니다.

---

### 1단계: 프로젝트 기획 및 방향 설정
*   **Q:** "@[프롬프트.md] 파일 내용으로 전문적인 프로그램 개요서를 Markdown 형식으로 '프로그램 개요서.md'로 만들어 주세요."
*   **Q:** "@프로그램개요서.md 프로그램 개요서를 기반으로 PRD(Product Requirements Document)를 작성해 주세요"

---

### 2단계: 개발 및 구현 (Sprints 1-3)
*   **Q:** "Sprint 1: 환경 설정 및 API 연동 진행해주세요"
*   **Q:** "Sprint 2: 프론트엔드 UI 개발 진행해주세요"
*   **Q:** "3단계: 백엔드 로직 구현** (완료 목표: Sprint 3) 진행해주세요"

---

### 3단계: 통합, 배포 및 실행 가이드
*   **Q:** "4단계: 통합 및 배포** (완료 목표: Sprint 4) 진행해주세요"
*   **Q:** "run / shutdown 하는 방법 '실행가이드.md' 파일로 만들어 주세요."

---

### 4단계: 오류 진단 및 트러블슈팅 (Hotfixes)
*   **Q:** "서버와 통신하는 중 오류가 발생했습니다. 원인 및 해결 방법 알려주세요."
*   **Q:** "GROQ_API_KEY 정상적으로 연동 되나요. 여전히 서버와 통신하는 중 오류가 발생 했습니다. 메세지 발생 합니다. 서버 ."
*   **Q:** "JSON Parsing Error: ParseError: KaTeX parse error: Can't use function '$' in math mode at position 1: $̲2$ ... (중략) ... Malformed Response Body: {\"result\": \"$2$\", \"explanation\": [...], \"similar_problem\": \"$2+2$\", \"similar_answer\": {...}} 오류 수정해주세요."
---

### 5단계: 문서화 마무리
*   **Q:** "질의 문장을 요약 하지 말고 ' 질문.md' 파일 만들어주세요."
